Designing and Creating On-board
PCB with STM32F4CCU6-Based
for PCR System

Achieve Skills: PCB Design, Schematic Design, Component Consideration, and PCB Layout rules.

The PCB is a system is PCR (Polymerase Chain Reaction), used to amplify DNA. This PCB contains several modules integrated as supplementary components in the PCR system. The on-board PCB design aims for spatial efficiency within a compact-sized device. Thus, the necessary modules do not require a new PCB. For example, the PCB is combine the STM32 microcontroller with an ADC converter. If we use both modules separately, the PCB system would not be compact and would still require connecting cables. Therefore, the PCB is integrated all these modules into a single PCB. The PCB consists of several modules including the STM32F401CCU6, MAX31865, BTS7960, and LM2596.
